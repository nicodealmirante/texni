# bo_texcno\nBot de WhatsApp con Baileys que reenvía solo mensajes con imagen y texto, aplicando un 15% extra a los precios.\n\n## Instalación\n```bash\ngit clone https://github.com/tuusuario/bo_texcno.git\ncd bo_texcno\nnpm install\ncp .env.example .env\n```\n\n## Configuración\nEditá `.env` con los IDs de los chats de origen y destino.\n\n## Ejecución\n```bash\nnode index.js\n```\nEscaneá el QR en consola con tu WhatsApp para vincular.\n\n## Deploy en Railway\nSubí este proyecto a un repo privado y conectalo en https://railway.app/new\n
